Terminal - Cheatsheet

  • Basics

  • / (forward Slash)

    Top level directory

  • . (Single Period)

    Current directory

  • .. (Double Period)

    Parent directory

  • ~ (Tilde)

    Home directory

  • exit

    Das aktuelle Terminal beenden/verlassen.

  • Clear

  • Pws

    Print Working Directory, Show your working directory

  • Lists Commands

  • ls

    list:Display the name of files and subdirectories in the directory

  • ls -1

    List in einer Spalte

  • ls -l

    List - long format ncludes file mode, owner and group name, date and time file was modified, pathname, and more

  • ls -a

    List mit versteckten Dateien

  • File and Directory Management

  • cd

    Home directory

  • cd [folder]

    Change directory, e.g. cd Documents

  • cd..

    Change directory zurück, move up to the parent directory

  • cd../..

    Move up two levels

  • cd ~

    Zurück zum User-Ordner mit Tilde (Alt N).

  • mkdir

    Make Directory

  • mkdir "Directory Name"

    Create a folder with a space in the filename

  • rm

    Delete a file (This deletes the file permanently; use with caution.)

  • touch

    Eine Datei erstellen

  • mv

    Datei verschieben

  • cp

    Datei Kopieren
